# 2026-05-14 — S2-3: Incident Response Plan (24-hr Breach Notification)

## Why

- TPRA submission requires a credible incident response plan demonstrating that BASELINE
  meets the 24-hour breach notification requirement from RCW 19.255.010 and the
  contractual obligation to notify KCM IT of incidents affecting KCM data
- KCM IT security review (S3-5) requires the plan exists before sign-off
